## Deskripsi Fitur Aplikasi
 
Aplikasi ini menampilkan halaman utama yang berisi daftar produk beserta kategori terkait. Melalui `HomeController`, seluruh data produk diambil bersama relasi kategorinya dan ditampilkan pada halaman **Index**. Selain itu, aplikasi juga menyediakan halaman **detail produk** sehingga pengguna dapat melihat informasi dari satu produk tertentu secara lebih lengkap.
 
Dari sisi manajemen data, aplikasi ini telah memiliki fitur **pengelolaan kategori** yang cukup lengkap. Melalui `CategoryController`, pengguna dapat melihat daftar kategori, menambahkan kategori baru, mengedit kategori yang sudah ada, serta menghapus kategori. Terdapat juga validasi sederhana yang membatasi agar nama kategori tidak boleh menggunakan nilai `"Test"`.
 
Untuk pengelolaan produk, aplikasi ini sudah mendukung beberapa fungsi utama seperti menampilkan daftar produk, menampilkan relasi antara produk dan kategorinya, membuat produk baru, serta memilih kategori saat proses penambahan produk. Pada `ProductController`, data produk diambil menggunakan `Include(p => p.Category)` agar informasi kategori dapat ditampilkan bersamaan. Saat proses create, daftar kategori dikirim ke view melalui `ViewBag.CategoryList`. Namun, berdasarkan controller yang tersedia saat ini, fitur **edit produk** dan **hapus produk** masih belum terlihat diimplementasikan.
 
Pada sisi keranjang belanja, aplikasi ini sudah mendukung alur dasar e-commerce, yaitu menambahkan produk ke **cart** dan melihat isi keranjang. Ketika pengguna menambahkan barang, data cart disimpan ke dalam **session** dalam bentuk JSON dengan key `ShoppingCart`. Selanjutnya, `CartController` akan membaca data session tersebut, mencocokkan `ProductId` dengan data produk di database, menghitung jumlah item (`quantity`), serta menjumlahkan total harga ke `OrderTotal`. Dengan demikian, aplikasi ini sudah dapat menjalankan alur dasar seperti **memilih produk → menambahkan ke keranjang → melihat isi keranjang dan total belanja**.
